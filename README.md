# 广西文旅网 - 广西旅游文化展示平台

一个展示广西壮族自治区旅游文化的单页网站应用。

## 项目简介

本项目是一个展示广西旅游资源和文化特色的网站，包含以下主要板块：

- **首页** - 广西旅游概览和快速导航
- **八桂风光** - 广西著名景点展示（漓江山水、阳朔西街、龙脊梯田等）
- **文化解码** - 广西民族文化、非遗瑰宝、民俗节庆等
- **风物档案** - 广西特色物产、美食、手工艺品等
- **旅游指南** - 互动地图、景点推荐、旅游攻略
- **在线咨询** - AI智能助手黄小西提供旅游咨询服务

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript** - 交互功能
- **Font Awesome** - 图标库

## 功能特点

1. **响应式设计** - 适配不同屏幕尺寸
2. **滚动动画** - 页面内容随滚动逐渐显示
3. **AI智能助手** - 黄小西提供智能旅游咨询
4. **互动地图** - 广西景点、美食、酒店标记
5. **轮播展示** - 景点图片轮播展示
6. **导航跳转** - 支持页面间快速跳转

## 部署方式

### GitHub Pages 部署

1. 将项目上传到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 访问 `https://<username>.github.io/<repository-name>/`

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/<username>/<repository-name>.git

# 进入项目目录
cd <repository-name>

# 使用浏览器打开
# 直接双击 guangxi-travel.html 或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```
